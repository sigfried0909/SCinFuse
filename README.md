# ScinFuse

ScinFuse is an educational science website developed for a DOST website development competition. The platform was created to provide interactive and accessible science learning materials for students.

1st Place — Educational Category, DOST Website Development Competition

This project was my first experience in website development and became the foundation of my interest in software and web technologies.

---

## Features

- Interactive science learning content
- Responsive user interface
- Organized educational modules
- Student-friendly design
- Integrated backend and database system

---

## Technologies Used

- HTML
- CSS
- JavaScript
- PHP
- MySQL
- Bootstrap

## My Role

- Developed backend functionalities

- Contributed to frontend implementation
- Performed testing and debugging
- Collaborated with the development team

---

## Project Background

ScinFuse was developed during a DOST-sponsored website development competition focused on educational technology. Despite having no formal IT background at the time, I participated in building the system’s backend, database, and parts of the frontend together with my team.

The project won 1st Place in the Educational Category and became a major milestone in my journey into software and web development.
